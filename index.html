▸ How This Works — Model & Data Sources
Overview
This dashboard uses a machine learning ensemble model trained on 17 years of historical weather data (2009-2026) to predict the probability of fog occurring at each airport during the next morning. The model runs every 15 minutes, scoring real-time weather observations against historical fog patterns.
ML Model
An ensemble of three algorithms — Gradient Boosting, Random Forest, and Logistic Regression — weighted by their individual performance. The model uses 142 features including temperature, dewpoint, humidity, wind, pressure, cloud cover, precipitation, TAF forecasts, and conditions at nearby airports. Each algorithm votes on the fog probability and the weighted average produces the final score.
Data Sources (pulled every 15 minutes)
• METAR observations — Real-time airport weather from the primary station and 4-5 nearby airports, via Iowa Environmental Mesonet (IEM ASOS). Includes temperature, dewpoint, wind, visibility, ceiling, weather phenomena, and pressure.
• TAF forecasts — Terminal Aerodrome Forecasts from IEM, parsed for fog/mist mentions, visibility restrictions, and ceiling forecasts across multiple time periods.
• NWS Forecast — National Weather Service overnight low temperature, wind, and conditions from the point forecast API. Also pulls hourly forecasts for the model-rescored timeline.
• NWS Alerts — Active fog advisories, dense fog advisories, and other weather alerts for the station's forecast zone.
• NDBC Buoy Data — Water temperature and air temperature from nearby ocean/river buoys (Astoria, Buoy 46029) for water-air temperature differential analysis.
• Nearby Airport Consensus — Current fog/mist status at surrounding airports to detect regional fog patterns.
Key Features the Model Considers
• Lead-time observations — Conditions at 12h, 8h, and 4h before target morning (evening, midnight, late night)
• Dew point depression trends — How fast the temp-dewpoint gap is closing overnight
• TAF consensus — How many nearby airport TAFs forecast fog/mist
• Wind patterns — Calm winds favor fog; gorge winds (easterly at Troutdale) indicate specific patterns
• Pressure trends — High pressure with clear skies favors radiation fog
• Fog persistence — Whether fog occurred yesterday or in the last 3 days
• Seasonality — Month-based fog frequency from 15 years of climatology
• Classic fog setup — High humidity (>85%), calm wind (<5kt), small dew dep (<4°F)
Hourly Risk Timeline
The timeline re-scores the ML model at each future hour using NWS hourly forecast data (projected temperature, dewpoint, humidity, wind) combined with the current TAF and nearby airport data. Each bar represents a real model prediction, not an estimate. Shows "● ML Model" when using actual model scores.
Accuracy & Limitations
• Model AUC ~0.956 on historical validation data
• Best accuracy after 6 PM when evening observations are available
• Earlier runs use training-set median values for unavailable features
• The model predicts probability, not certainty — use alongside standard weather briefing
• Not a substitute for official NWS products or pilot weather briefings
